# 天然沐浴用品電商網站

<br />

> 作品介紹
+ 這是我的作品，一個以天然沐浴用品為概念的電商購物網站，這個作品的概念，是因為我本身有皮膚敏感的問題，以本身的問題所構想出來的作品

<br />

> 使用技術
+ Vue 2 
+ Bootstrap 4
+ Sass
+ 串接 Restful Api
+ Aos 動畫套件

<br />

> 作品網址

https://because-hung.github.io/vue.js-project-web/#/
